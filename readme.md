#Summary
A bootstrap4 template that utilizes gulp to pipe required files into the public folder and serve a live view server.

## Contents
jquery
bootstrap4
popper.js
gulp
animate.css

### GULP
Pipes bs4 css/js, jquery into public/assets/..
Starts server, listens for changes.
